Abra o Git Bash:


Execute o comando git version para verificar se a instalação foi realizada corretamente:


Execute o comando git config --global user.name "SEU_NOME_DE_USUÁRIO_DO_GITHUB":


Execute o comando git config --global user.email "SEU_EMAIL_DO_GITHUB":


Execute os comandos git config user.name e git config user.email para verificar se o seu nome de usuário e e-mail foram configurados corretamente:
