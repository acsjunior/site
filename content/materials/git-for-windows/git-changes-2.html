Comandos utilizados

  • git diff: exibe as alterações realizadas
  • git restore xxxx: desfaz as alterações realizadas antes de enviar para a staging area (xxxx é o nome do arquivo)
  • git restore –staged *: desfaz o envio das alterações para a staging area
  • git reset yyyy: desfaz as alterações salvas (yyyy é o id resumido do commit)
