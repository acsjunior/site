O que é Git?

Git é um sistema de controle de versões. Tem como objetivo documentar e armazenar todas as alterações de um projeto.

O que é GitHub?

GitHub é uma rede social de projetos que tem como objetivo hospedar remotamente e compartilhar projetos. Além disso, permite o trabalho em equipe, tirar dúvidas, relatar problemas e etc.

Vantagens

Trabalhar com Git e GitHub nos permite realizar o backup do nosso projeto, recuperar versões anteriores, unificar todas as alterações realizadas pela equipe e construir um portfólio.

Comandos utilizados

  • git init: inicializa o repositório git
  • git status: verifica o status do repositório
  • git add –all: adiciona todas as alterações na staging area
  • git commit -m "mensagem": salva as alterações
